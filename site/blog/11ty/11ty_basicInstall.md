```bash
mkdir 11ty_sample
cd 11ty_sample
npm init -y
npm install --save-dev @11ty/eleventy
```
위에 과정을 살펴보면 11ty_sample 디렉토리를 만들고 node 프로젝트 구성에 11ty를 포함시키는 명령어 입니다.
